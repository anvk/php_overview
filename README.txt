This repository contains three demos for "PHP Overview" community meeting held on Jan 21, 2013.

    * Hello world
      A simply start to show how to embed the php code into html source.
      Access via hello_world.php
      
    * PHP info demo
      Demostrates the power of phpinfo() function.
      Access via phpinfo.php
      
    * Create a wish list of the willing-to-travel places
      Demostrates connecting mysql database, posting form data and establishing sessions.
      Access via login.php
